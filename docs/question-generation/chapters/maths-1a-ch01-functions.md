## Weighting rationale
- **Cartesian Product and Relations (10):** Basic set operations, product sizes, binary relations.
- **Types of Functions (20):** Identifying injectivity, surjectivity, bijectivity for different domains/codomains.
- **Counting and Equality of Functions (15):** Equality conditions, counting injections/surjections/bijections using permutations and combinations.
- **Composition of Functions (25):** Core operational calculus of composing functions, solving equations like $f(g(x)) = g(f(x))$.
- **Inverse Functions (25):** Solving for $f^{-1}(x)$, checking invertibility, composition of inverses.
- **Domain and Range of Real Functions (30):** The heavy algebraic core (finding domains of rational functions, square roots, composite functions).
- **Special Functions and Even-Odd Nature (25):** Modulus, GIF, Signum, even/odd decomposition, algebra of these functions.
