Example Physics · Motion in a Straight Line · Kinematics + energy Medium
A ball is thrown vertically upward with a speed of $20\,\text{m/s}$ from the top of a tower of height $25\,\text{m}$. Taking $g = 10\,\text{m/s}^2$ and ignoring air resistance, with what speed does it strike the ground?
  1. A$20\,\text{m/s}$
  2. B$30\,\text{m/s}$✓ correct
  3. C$25\,\text{m/s}$
  4. D$35\,\text{m/s}$
⚡ Quick Method
  1. Only gravity acts, so the landing speed depends just on the start speed and the total drop — the trip up and back cancels out.
  2. Use $v^2 = u^2 + 2gh$ with $u = 20$, $h = 25$, $g = 10$: $v^2 = 400 + 500 = 900$.
  3. $v = \sqrt{900} = 30\,\text{m/s}$ → option B.
🔬 Full Solution
Mechanical energy is conserved because only gravity does work, so the final speed is fixed by the launch speed and the net drop — independent of the path. The clean tool is the constant-acceleration relation $v^2 = u^2 + 2as$.
  1. Set up the situation and a sign convention.The only force is gravity, so the acceleration is constant at $g = 10\,\text{m/s}^2$ pointing down. Take downward as positive throughout.
  2. See why the upward part doesn't change the answer.The ball rises, momentarily stops, and falls back to its launch height with the same $20\,\text{m/s}$ it started with. So the problem is equivalent to dropping the ball from the tower top with an initial downward speed of $20\,\text{m/s}$.
  3. List the knowns for the fall.Initial speed $u = 20\,\text{m/s}$, displacement to the ground $s = +25\,\text{m}$, acceleration $a = +10\,\text{m/s}^2$.
  4. Apply the kinematic equation.Substituting into $v^2 = u^2 + 2as$ gives $v^2 = (20)^2 + 2(10)(25)$.
  5. Do the arithmetic.$v^2 = 400 + 500 = 900\,\text{m}^2/\text{s}^2$.
  6. Take the square root and keep the physical root.$v = \sqrt{900} = 30\,\text{m/s}$; speed is positive, so $v = 30\,\text{m/s}$.
Answer: $v = 30\,\text{m/s}$ — option B.
