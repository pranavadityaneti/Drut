A ball is thrown vertically upward at $20\,\text{m/s}$ from the top of a $25\,\text{m}$ tower. With $g=10\,\text{m/s}^2$ and no air resistance, with what speed does it strike the ground?  (Answer: 30 m/s, option B)
Variant A
Numbered steps + bold headers
Most scannable; feels like a checklist (the "1-2-3" you want to avoid).
Only gravity acts, so mechanical energy is conserved and the final speed depends only on the launch speed and the net drop. Use $v^2 = u^2 + 2as$.
  1. Set a sign convention.Acceleration is constant at $g=10\,\text{m/s}^2$ downward; take downward as positive.
  2. Ignore the upward leg.The ball returns to launch height at the same $20\,\text{m/s}$, so it's equivalent to dropping it from the top at $20\,\text{m/s}$.
  3. Apply $v^2=u^2+2as$.$v^2 = 20^2 + 2(10)(25) = 400 + 500 = 900$.
  4. Take the root.$v=\sqrt{900}=30\,\text{m/s}$.
Answer: $30\,\text{m/s}$ — option B.

Only gravity acts, so mechanical energy is conserved — the final speed depends only on the launch speed and the net drop. Use $v^2 = u^2 + 2as$.

Only gravity acts on the ball, so its acceleration is constant at $g=10\,\text{m/s}^2$ downward; take downward as positive. The upward leg doesn't change the outcome — the ball returns to its launch height with the same $20\,\text{m/s}$ it left with, so the motion is equivalent to dropping it from the tower top at $20\,\text{m/s}$.

With $u=20\,\text{m/s}$, $s=25\,\text{m}$ and $a=10\,\text{m/s}^2$, the kinematic relation gives

$$v^2 = u^2 + 2as = 20^2 + 2(10)(25) = 400 + 500 = 900.$$

Taking the positive root, $v=\sqrt{900}=30\,\text{m/s}$, so the ball strikes the ground at $30\,\text{m/s}$ — option B.

Answer: $30\,\text{m/s}$ — option B.
