Q1Mathematics · Vectors · answer verified ✓Hard
Let the position vectors of the vertices of a triangle ABC be $\vec{a}, \vec{b}, \vec{c}$. If on the plane of the triangle, P is a point having position vector $\vec{x}$ such that $\vec{x}\cdot(\vec{c}-\vec{b}) = \vec{a}\cdot\vec{c} - \vec{a}\cdot\vec{b}$ and $\vec{x}\cdot(\vec{a}-\vec{c}) = \vec{a}\cdot\vec{b} - \vec{b}\cdot\vec{c}$, then for the triangle ABC, P is the
  1. ACentroid
  2. BCircumcentre
  3. CIncentre
  4. DOrthocentre✓
⚡ Quick Method
  1. Rewrite each condition by moving the right-hand side to the left.
  2. Then $(\vec{x}-\vec{a})\cdot(\vec{c}-\vec{b})=0$ and $(\vec{x}-\vec{b})\cdot(\vec{a}-\vec{c})=0$.
  3. So $AP\perp BC$ and $BP\perp AC$; their intersection is the orthocentre.
🔬 Full Solution 3 steps
Use the dot-product conditions to identify perpendicularity with the opposite sides of the triangle.
From $\vec{x}\cdot(\vec{c}-\vec{b})=\vec{a}\cdot\vec{c}-\vec{a}\cdot\vec{b}=\vec{a}\cdot(\vec{c}-\vec{b})$, we get $(\vec{x}-\vec{a})\cdot(\vec{c}-\vec{b})=0$. This means the vector $\overrightarrow{AP}$ is perpendicular to $\overrightarrow{BC}$, so $P$ lies on the altitude from $A$.
$$(\vec{x}-\vec{a})\cdot(\vec{c}-\vec{b})=0$$
Similarly, $\vec{x}\cdot(\vec{a}-\vec{c})=\vec{a}\cdot\vec{b}-\vec{b}\cdot\vec{c}=\vec{b}\cdot(\vec{a}-\vec{c})$, hence $(\vec{x}-\vec{b})\cdot(\vec{a}-\vec{c})=0$. Therefore $\overrightarrow{BP}\perp\overrightarrow{AC}$, so $P$ also lies on the altitude from $B$.
$$(\vec{x}-\vec{b})\cdot(\vec{a}-\vec{c})=0$$
The point where two altitudes of a triangle meet is the orthocentre. Hence $P$ is the orthocentre of triangle $ABC$.
Answer: D. Orthocentre

  1. Teflon is PTFE, i.e. polytetrafluoroethylene.

  3. So the monomer is tetrafluoroethylene.
🔬 Full Solution 2 steps
Teflon is the trade name for polytetrafluoroethylene, so we identify the alkene that polymerizes to give the PTFE chain.
Polytetrafluoroethylene is obtained by addition polymerization of tetrafluoroethylene, whose molecule is $CF_2=CF_2$.
$$nCF_2=CF_2 \rightarrow (-CF_2-CF_2-)_n$$

Q3Chemistry · Equilibrium · answer verified ✓Easy
What is the pH of $0.001$ M NaOH solution?
  1. A$3$
  2. B$11$✓
  3. C$14$
  4. D$7$
⚡ Quick Method
  1. NaOH fully dissociates, so $[\mathrm{OH}^-]=0.001\,\mathrm{M}=10^{-3}\,\mathrm{M}$.
  2. Compute $\mathrm{pOH}=-\log(10^{-3})=3$.
  3. Use $\mathrm{pH}+\mathrm{pOH}=14$, so $\mathrm{pH}=11$.
🔬 Full Solution 3 steps
Use complete dissociation of strong base and the relation between pH and pOH at $25^\circ\mathrm{C}$.
NaOH is a strong base, so it dissociates completely in water: $\mathrm{NaOH} \rightarrow \mathrm{Na}^+ + \mathrm{OH}^-$. Therefore, the hydroxide ion concentration is $[\mathrm{OH}^-]=0.001\,\mathrm{M}=10^{-3}\,\mathrm{M}$.
$$[\mathrm{OH}^-]=10^{-3}\,\mathrm{M}$$
Now find pOH using $\mathrm{pOH}=-\log[\mathrm{OH}^-]$. Substituting gives $\mathrm{pOH}=-\log(10^{-3})=3$.
$$\mathrm{pOH}=-\log(10^{-3})=3$$
At $25^\circ\mathrm{C}$, $\mathrm{pH}+\mathrm{pOH}=14$. So $\mathrm{pH}=14-3=11$.
$$\mathrm{pH}=14-3=11$$
Answer: The pH of the $0.001\,\mathrm{M}$ NaOH solution is $11$, so option B is correct.

Q5Mathematics · Applications of Derivatives · answer verified ✓Medium
If the normal drawn at the point P on the curve $y = x\log x$ is parallel to the line $2x - 2y = 3$, then $P =$
  1. A$(e, e)$
  2. B$\left(\frac{1}{e}, \frac{-1}{e}\right)$
  3. C$\left(\frac{1}{e^2}, \frac{-2}{e^2}\right)$✓
  4. D$(e^3, 3e^3)$
⚡ Quick Method
  1. Line $2x-2y=3$ has slope $1$. So the normal slope is $1$, hence tangent slope is $-1$.
  2. For $y=x\log x$, $\frac{dy}{dx}=\log x+1$. Set $\log x+1=-1$.
  3. Then $\log x=-2\Rightarrow x=e^{-2}$ and $y=x\log x=e^{-2}(-2)=-\frac{2}{e^2}$.
🔬 Full Solution 3 steps
Use the fact that a normal is perpendicular to the tangent, so its slope determines the derivative at the point on the curve.
The given line $2x-2y=3$ can be written as $y=x-\frac{3}{2}$, so its slope is $1$. Since the normal is parallel to this line, the normal slope at $P$ is also $1$, which makes the tangent slope $-1$.
$$m_{\text{normal}}=1,\quad m_{\text{tangent}}=-1$$
For the curve $y=x\log x$, the derivative is $\frac{dy}{dx}=\log x+1$. At the point of tangency, this must equal the tangent slope $-1$.
$$\frac{dy}{dx}=\log x+1=-1$$
So $\log x=-2$, giving $x=e^{-2}=\frac{1}{e^2}$. Substituting in $y=x\log x$, we get $y=e^{-2}(-2)=-\frac{2}{e^2}$. Therefore, $P=\left(\frac{1}{e^2},-\frac{2}{e^2}\right)$.
$$y=e^{-2}( -2 )=-\frac{2}{e^2}$$
Answer:
\left(\frac{1}{e^2},-\frac{2}{e^2}\right)
Q6Mathematics · Definite Integration · answer verified ✓Medium
$\int_0^{\pi/2} \log|\tan x + \cot x|\,dx =$
  1. A$\pi\log 2$✓
  2. B$-\pi\log 2$
  3. C$\frac{\pi}{2}\log 2$
  4. D$2\pi\log 2$
⚡ Quick Method
  1. Use $\tan x+\cot x=\frac{1}{\sin x\cos x}$ for $0<x<\frac{\pi}{2}$.
  2. So the integrand is $-\log(\sin x)-\log(\cos x)$.
  3. Apply $\int_0^{\pi/2}\log(\sin x)\,dx=\int_0^{\pi/2}\log(\cos x)\,dx=-\frac{\pi}{2}\log 2$.
🔬 Full Solution 3 steps
Rewrite the expression in terms of $\sin x$ and $\cos x$, then use the standard definite integrals of $\log(\sin x)$ and $\log(\cos x)$ on $\left[0,\frac{\pi}{2}\right]$.
For $0<x<\frac{\pi}{2}$, both $\tan x$ and $\cot x$ are positive, so the absolute value is unnecessary. Also, \[\tan x+\cot x=\frac{\sin^2 x+\cos^2 x}{\sin x\cos x}=\frac{1}{\sin x\cos x}.\]
$$\tan x+\cot x=\frac{1}{\sin x\cos x}$$
Hence, \[\log(\tan x+\cot x)=-\log(\sin x)-\log(\cos x).\] Therefore, \[I=\int_0^{\pi/2}\log(\tan x+\cot x)\,dx=-\int_0^{\pi/2}\log(\sin x)\,dx-\int_0^{\pi/2}\log(\cos x)\,dx.\]
$$I=-\int_0^{\pi/2}\log(\sin x)\,dx-\int_0^{\pi/2}\log(\cos x)\,dx$$
Using the standard result \[\int_0^{\pi/2}\log(\sin x)\,dx=\int_0^{\pi/2}\log(\cos x)\,dx=-\frac{\pi}{2}\log 2,\] we get \[I=-\left(-\frac{\pi}{2}\log 2\right)-\left(-\frac{\pi}{2}\log 2\right)=\pi\log 2.\]
$$I=\pi\log 2$$
Answer: $\boxed{\pi\log 2}$
