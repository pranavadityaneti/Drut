Q1Physics · Chapter 5: Laws of Motion · APPROVED ✓ (live) · static friction with connected bodiesHard
A $4\,\mathrm{kg}$ block $A$ on a rough horizontal table is tied by a light inextensible string over a smooth pulley to a hanging block $B$ of mass $3\,\mathrm{kg}$. The coefficient of static friction between $A$ and the table is $0.3$. A horizontal force $F$ is applied to $A$ away from the pulley. If $g=10\,\mathrm{m\,s^{-2}}$, the system can stay at rest only for which interval of $F$?
  1. A$18\,\mathrm{N} \le F \le 42\,\mathrm{N}$✓
  2. B$0 \le F \le 42\,\mathrm{N}$
  3. C$18\,\mathrm{N} \le F \le 30\,\mathrm{N}$
  4. D$12\,\mathrm{N} \le F \le 30\,\mathrm{N}$
⚡ Quick Method
  1. Find the string tension from block $B$ and the maximum static friction on block $A$.
  2. Write the equilibrium condition for $A$ for both possible impending motions.
  3. Combine the lower and upper bounds to get the allowed interval of $F$.
🔬 Full Solution 5 steps
Since the question asks when the system can remain at rest, use static equilibrium for both blocks. The key is that static friction can act in either direction, so the rest condition becomes a force window.
Because the system is at rest, block $B$ has zero acceleration, so the string tension must balance its weight. Thus $T = m_B g = 3 \times 10\,\mathrm{N} = 30\,\mathrm{N}$.
$$T = m_B g = 3 \times 10 = 30\,\mathrm{N}$$
For block $A$, the normal reaction is $N = 4g = 40\,\mathrm{N}$. Hence the largest possible static friction is $f_{s,\max} = \mu_s N = 0.3 \times 40 = 12\,\mathrm{N}$.
$$f_{s,\max} = \mu_s N = 0.3 \times (4 \times 10) = 12\,\mathrm{N}$$
If $F$ is too small, block $B$ tends to move downward and block $A$ tends to move toward the pulley. Then friction on $A$ acts away from the pulley, so equilibrium requires $T = F + f_s$ with $f_s \le 12\,\mathrm{N}$.
$$T = F + f_s \Rightarrow F \ge 30 - 12 = 18\,\mathrm{N}$$
If $F$ is too large, block $A$ tends to move away from the pulley. Then friction reverses direction and acts toward the pulley, so equilibrium requires $F = T + f_s$ with $f_s \le 12\,\mathrm{N}$.
$$F = T + f_s \Rightarrow F \le 30 + 12 = 42\,\mathrm{N}$$
Both limits must hold together, so the system can remain at rest only when $18\,\mathrm{N} \le F \le 42\,\mathrm{N}$. This corresponds to option A.
$$18\,\mathrm{N} \le F \le 42\,\mathrm{N}$$
Answer: The permissible range is $18\,\mathrm{N} \le F \le 42\,\mathrm{N}$, so the correct option is A.
Q2Physics · Chapter 5: Laws of Motion · staged · glancing elastic collisionsHard
On a smooth horizontal table, a particle of mass $m$ moving with speed $u$ collides elastically with a stationary particle of mass $2m$. Immediately after the impact, the lighter particle moves at $90^\circ$ to its original direction of motion. What is its speed?
  1. A$\frac{u}{\sqrt{3}}$✓
  2. B$\frac{u}{2}$
  3. C$\frac{u}{\sqrt{2}}$
  4. D$\frac{2u}{\sqrt{3}}$
⚡ Quick Method
  1. Take the initial motion as the $x$-axis and the new direction of the lighter particle as the $y$-axis.
  2. Use momentum conservation in $x$ and $y$ to get the heavier particle's components.
  3. Apply kinetic-energy conservation, simplify, and solve for the lighter particle's speed.
🔬 Full Solution 5 steps
Because the table is smooth, there is no external horizontal impulse, so momentum is conserved in both perpendicular directions. Since the collision is elastic, kinetic energy is also conserved, which closes the system.
Take the initial motion along the $x$-direction. Let the lighter particle's speed after collision be $v$ along $y$, and let the heavier particle have components $V_x$ and $V_y$.
Conservation of momentum along $x$ gives the horizontal component of the heavier particle.
$$mu = 2mV_x \Rightarrow V_x = \frac{u}{2}$$
Conservation of momentum along $y$ uses the fact that the initial $y$-momentum is zero, so the final $y$-components must cancel.
$$0 = mv + 2mV_y \Rightarrow V_y = -\frac{v}{2}$$
Because the collision is elastic, kinetic energy is conserved. Write the final energy using the components of the heavier particle and substitute the two component relations.
$$\frac{1}{2}mu^2 = \frac{1}{2}mv^2 + \frac{1}{2}(2m)\left(V_x^2 + V_y^2\right)$$
Now substitute $V_x = \frac{u}{2}$ and $V_y = -\frac{v}{2}$, then simplify to get the speed of the lighter particle.
$$\frac{1}{2}u^2 = \frac{1}{2}v^2 + \frac{u^2}{4} + \frac{v^2}{4} \Rightarrow u^2 = 3v^2 \Rightarrow v = \frac{u}{\sqrt{3}}$$
Answer: The lighter particle's speed is $\frac{u}{\sqrt{3}}$, so the correct option is A.
Q3Physics · Chapter 5: Laws of Motion · staged · friction on horizontal surface with an inclined pulling forceMedium
A $5\,\text{kg}$ block slides on a horizontal rough floor when a $25\,\text{N}$ force acts on it at an angle of $37^\circ$ above the floor. If the coefficient of kinetic friction is $0.3$ and $g=10\,\text{m s}^{-2}$, with $\cos 37^\circ = 0.8$ and $\sin 37^\circ = 0.6$, what is its acceleration?
  1. A$1.9\,\text{m s}^{-2}$✓
  2. B$2.5\,\text{m s}^{-2}$
  3. C$4.0\,\text{m s}^{-2}$
  4. D$1.0\,\text{m s}^{-2}$
⚡ Quick Method
  1. Resolve the pull into horizontal and vertical parts.
  2. Find $N$, then compute friction as $\mu N$.
  3. Use net horizontal force $= ma$ to get the acceleration.
🔬 Full Solution 5 steps
Apply Newton's second law separately along vertical and horizontal directions. The upward component of the pull reduces the normal reaction, so friction must be calculated from the reduced normal before finding the net horizontal force.
Split the applied force into components. The horizontal part drives the block, while the vertical part lifts it slightly and changes the normal reaction.
$$F_x = 25 \cos 37^\circ = 20\,\text{N}, \quad F_y = 25 \sin 37^\circ = 15\,\text{N}$$
There is no vertical acceleration, so the normal reaction equals the weight minus the upward component of the pull.
$$N = mg - F_y = 5 \times 10 - 15 = 35\,\text{N}$$
Kinetic friction acts opposite the motion and has magnitude $\mu N$.
$$f = \mu N = 0.3 \times 35 = 10.5\,\text{N}$$
The net horizontal force is the driving component minus friction. Dividing by mass gives the acceleration.
$$ma = F_x - f = 20 - 10.5 = 9.5\,\text{N}$$
Now solve for $a$.
$$a = \frac{9.5}{5} = 1.9\,\text{m s}^{-2}$$
Answer: The acceleration is $1.9\,\text{m s}^{-2}$, so option A is correct.
Q4Physics · Chapter 5: Laws of Motion · staged · mixedMedium
A $5\,\text{kg}$ block is pulled on a rough horizontal floor by a $25\,\text{N}$ force acting at $37^\circ$ above the horizontal. If $\mu_k=0.2$ and $g=10\,\text{m s}^{-2}$, the block's acceleration while it slides is
  1. A$2.6\,\text{m s}^{-2}$✓
  2. B$2.0\,\text{m s}^{-2}$
  3. C$1.6\,\text{m s}^{-2}$
  4. D$1.4\,\text{m s}^{-2}$
⚡ Quick Method
  1. Split the pull into $25\cos37^\circ$ and $25\sin37^\circ$.
  2. Find $N=mg-25\sin37^\circ$ and friction $f=\mu_kN$.
  3. Use $ma=25\cos37^\circ-f$ to get $a$.
🔬 Full Solution 5 steps
Resolve the pull into components and apply Newton's second law along each axis. Friction depends on the reduced normal reaction, so the vertical balance must be done first.
Split the applied force into horizontal and vertical parts. The vertical part lifts the block slightly, so it lowers the normal reaction and therefore the friction.
$$F_x=25\cos37^\circ=20\,\text{N},\qquad F_y=25\sin37^\circ=15\,\text{N}$$
Since the block has no vertical acceleration, the upward forces balance the weight. So the normal reaction is the weight minus the upward component of the pull.
$$N=mg-F_y=5\times10-15=35\,\text{N}$$
Now compute the kinetic friction. It opposes the motion along the floor.
$$f_k=\mu_kN=0.2\times35=7\,\text{N}$$
Apply Newton's second law along the horizontal direction. The net force is the horizontal pull minus friction.
$$ma=F_x-f_k\Rightarrow 5a=20-7=13$$
Hence the acceleration is $13/5=2.6\,\text{m s}^{-2}$. So option A is correct.
$$a=\frac{13}{5}=2.6\,\text{m s}^{-2}$$
Answer: The acceleration is $2.6\,\text{m s}^{-2}$, which is option A.
